## 🚀 SideQuest — local services marketplace
Two installable web apps sharing one backend, live in production on custom domains:

- 🏠 **Customer app:** https://joinsidequest.us
- 👷 **Quester (worker) app:** https://pro.joinsidequest.us

**What it does:** customers post home & task jobs ("Quests"); independent workers accept and complete them. Payments and worker payouts run through **Stripe Connect**, with identity verification, in-app messaging, ratings & reviews, and push notifications.

**Built with:** React · TanStack Start (SSR) · TypeScript · Vite · Nitro · Tailwind CSS · Supabase (Postgres, Auth, Storage, Row-Level Security) · Stripe Connect · Vercel · installable PWA
